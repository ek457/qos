Python 3 安装帮助

前言

对于Quarter OS，需要的最低Python版本为 Python 3.10。

如果允许，我们更建议您使用 Python 3.12 版本。因为这和Quarter OS的开发环境一致。

对于 Windows 用户

Windows端最低支持Windows 8.1，但Quarter OS需要的最低版本为Windows 10。

① 前往 Python 官网 或 阿里云镜像源 下载 exe 或 msi 安装包。

下载完成后直接安装即可。

注意：安装界面中，请勾选 Add Python 3.x to PATH 选项。

② 通过winget命令安装：

# 查找Python版本
winget search Python.Python
# 比如要下载Python3.12
winget install --id Python.Python.3.12 --source winget --scope machine

注意：本方法仅适用于Windows 10/11，且需要先安装 winget (通常只针对于企业版或没有软件包安装程序的用户)。

③ 直接在Microsoft Store搜索 Python 3 安装。

注意：本方法仅适用于Windows 10/11，且系统已带有Microsoft Store的用户。

对于 macOS 用户

macOS端最低支持 macOS 10.9 Mavericks，但Quarter OS主要开发环境为Windows和Linux，所以在macOS上使用Quarter OS可能存在一些问题。

① 前往 Python 官网 或 阿里云镜像源 下载 dmg 安装包。

② 使用 Homebrew 安装 Python3：

# 因为macOS内置Python2，所以这里要安装"python3"
brew install python3

对于 Linux 用户

最低支持 glibc 2.17，若低于该版本可能需要自行编译。绝大部分常见的Linux发行版均满足此要求。

Debian/Ubuntu：

（最低支持：Debian 10 Buster、Ubuntu 20.04 LTS Focal）

sudo apt-get install python3

CentOS/Fedora：

（最低支持：CentOS 8 Stream、Fedora 32）

sudo dnf install python3

Arch Linux：

（最低支持：未知）

sudo pacman -S python

其他发行版：

请自行查找教程，并使用对应的软件包管理器安装Python 3。

Linux注意事项

绝大部分的Linux发行版中自带Python3，但通常版本过低或没有提供完整版。

比如 Ubuntu 24.04 LTS Noble 自带Python 3.12.3，但没有pip、venv等工具。

所以，我们需要安装完整版的Python 3。

# 更新软件源
sudo apt-get update
sudo apt-get upgrade

# 安装Python 3 完整版，比如安装Python3.12完整版
sudo apt-get install python3.12-full

# 验证安装成功
python3.12 -v

至此，Linux端的Python 3安装完毕。
