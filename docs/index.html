Quarter OS

Pro级的Python3伪系统

介绍

Quarter OS 是一款基于 Python 3 的伪操作系统，简称 QOS，由 ElofHew 开发。

（注：QOS是Quarter Operation System的缩写，与网络技术中的“QoS”（Quality of Service，IP网络服务质量保证）无关。）
