Quarter OS 快速使用指南

下载安装

下载QOS Release：GitHub Release

如果你想参与开发，可以直接下载源代码：GitHub Repository

（注意，可能遇到bug或不稳定的功能，请不要在生产环境中使用）

终端运行：git clone https://github.com/ElofHew/QOS.git

配置并启动QOS

① 安装Python 3.10及以上版本

请详阅 Python3 安装帮助

② 配置一个Python虚拟环境

请详阅 Python虚拟环境帮助

③ 激活虚拟环境

# Windows
# 例如虚拟环境名为 qosvenv
.\qosvenv\Scripts\Activate.ps1
# Linux/macOS
# 例如虚拟环境名为 qosvenv
source ./qosvenv/bin/activate

④ 在虚拟环境中安装依赖

找到本存储库根目录的 requirements.txt 文件，使用 pip 安装依赖：

pip install -r requirements.txt

如果你所在地区访问pypi官方源的速度较慢，可以尝试换源，详阅 pip换源帮助

⑤ 在虚拟环境中运行Quarter OS

注意，对于Windows系统，请在PowerShell中运行，而非cmd或其他命令行。 Linux/macOS用户请在系统自带的终端中运行。

进入Quarter OS文件夹，并在此处打开终端，输入以下命令：

# Windows
python qos.py
# Linux/macOS
python3 qos.py

然后 Quarter OS 就会启动。

开始使用

① OOBE阶段

在您第一次启动Quarter OS的时候，也许会卡在第一屏很长时间不继续，不用慌张，这是Quarter OS在联网获取广告等信息，缓存到数据目录，如果实在连接不到，会超时并跳过，继续下一步。

接下来，Quarter OS会进入到OOBE阶段。

OOBE（Out-of-Box Experience，即开箱即用），会在Quarter OS第一次启动时，进行系统的一些基本配置。

在OOBE阶段，你将进行以下步骤：

  • 阅读并同意Quarter OS的最终用户许可协议
  • 创建一个用户账户
  • 设置主机名称
  • 激活Quarter OS（详见文档：激活帮助）

完成OOBE后，Quarter OS会自动重启，进入到登录界面。

② 登录界面

登录界面会让你输入用户名和密码，校验通过后即可进入系统

③ 终端界面

登录成功后，Quarter OS会进入到终端界面，至此，你就可以开始使用Quarter OS了。

在终端界面，你可以输入命令，执行各种操作。关于受支持的命令，请参阅文档：命令帮助

内置账户密码

通常情况下，当你第一次启动Quarter OS，进入OOBE阶段，会让你创建一个用户名和密码，所以不需要用到内置的 root、admin、guest 等账户。但在这里也列出这些账户的密码以备用：
