Quarter OS 命令功能帮助

系统内置命令

（不需要参数的命令）

命令 描述
pwd 显示当前工作目录
cwd 显示真实的当前工作目录
help 显示可用命令列表
about 显示关于QOS的信息
version 显示QOS的系统版本
whoami 显示当前用户
time 显示当前时间
date 显示当前日期
sysinfo 显示系统信息
clear 清空控制台
eula 显示QOS最终用户许可协议
exit 退出QOS
reboot 重启QOS

（需要参数的命令）

命令 描述 用法
cat 显示文件内容 cat <文件名>
echo 在控制台上显示消息 echo <消息>
cp 复制文件或目录 cp <源文件/目录> <目标文件/目录>
mv 移动文件或目录 mv <源文件/目录> <目标文件/目录>
ls 显示目录中的文件和目录列表 ls <目录(空则当前目录)>
cd 更改当前目录 cd <目录>
touch 创建新文件 touch <文件名>
edit 编辑文件 edit <文件名>
mkdir 创建新目录 mkdir <目录名>
rename 重命名文件或目录 rename <源文件/目录> <目标文件/目录>
rm 删除文件或目录 rm <文件/目录>
ping 测试网络连接 ping <域名或IP地址>
down 下载文件 down <文件URL>

第三方软件包管理器

（Biscuit软件包管理器）

  • biscuit : Quarter OS官方直属软件包管理器。
  • bkt/bpm/pkg/pm : 调用Biscuit的易记别名。
命令 描述 用法
install 安装(本地)软件包 biscuit install <文件路径>
remove 卸载软件包 biscuit remove <软件包名>
search 搜索软件包 biscuit search <软件名关键字>
list 列出已安装的软件包 biscuit list
get 从软件仓库获取并安装软件包 biscuit get <软件包名> (<指定包版本>)
mirror 更改软件包镜像源 biscuit mirror <镜像源地址>

（Shizuku软件包管理器）

  • shizuku : PY OS Improved的软件包管理器，Quarter OS对其进行兼容。
  • szk : 调用Shizuku的易记别名。
命令 描述 用法
install 安装软件包 shizuku install <软件包名>
remove 卸载软件包 shizuku remove <软件包名>
list 列出已安装的软件包 shizuku list
run 启动Shizuku软件包 shizuku run <软件包名>
